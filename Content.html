How to start with IGCMS
  1. Prepare well-structured concept (document) of what you want to achieve. Focus on the structure and content (not only content).
    • Every concept has one and only one main heading (title).
    • Every heading should be followed by a summary (description) of the following sub/chapter.
    • The content should be “balanced”, meaning there should not be too little or too many elements (headings, paragraphs, list items) after each other.
    • Find more about creating structural contents online.
  2. Have a look at IGCMS Specification page.
    • Login: partner / igpartner
    • Make sure you understand the basic terminology first!
    • If you are lost, always RTFM :-)
  3. Import your document into IGCMS using ?import parameter.
    • Remember the imported file is stored separatedly to prevent unexpected results.
    • To use the content (or a part of it) copy the imported code into content document.
    • Or you can “join” the generated document with the content document using import attribute inside the content document.
  4. Enter administration using ?admin parameter for further modifications or use FTP access to modify files directly.
