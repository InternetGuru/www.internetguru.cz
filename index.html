Web Developer Internet Guru We develop and provide software and business model for long-term web evolution. We provide both technological and business background for collaborators including independent companies. We also research and educate.
Projects and Services We offer tools supporting own clients for independent collaborators. We also provide appropriate training for individual tools as well as services based on that tools.
Internet Guru Content Management System
IGCMS is an commercial web content management system optimised for effective web page creation. As our collaborator you can use the system in two different variants according to automatic updating.
Oh My Git Flow (OMGF)
OMGF is an open source command line tool available on GitHub. It supports file structure and content organisation following extended Git Flow branching model including version history. The model is suitable for any stability reliant projects (such as web services) and for large projects in general.
Bash Unit Testing Tool
Open source command line tool available on GitHub to support programmers in testing their commands.
Web Comparison Tool
Server command to watch and monitor changes within a given website. It is able to check for differences on demand or regularly sending the output to e-mail.
Available for our collaborators.
Web Content Analyser
Tool to check and evaluate websites tens of technological and content criteria. It serves to verify new websites as well as to evaluate current third party websites.
Available for our collaborators.
